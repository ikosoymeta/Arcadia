# Arcadia — Claude Web Editor

AI-powered web editor with Manus-style UI, real-time preview, VS Code-like code workspace, and built-in performance benchmarking.

## Features

- **Chat Interface** — Streaming conversations with Claude, markdown rendering, artifact extraction
- **Real-Time Preview** — Code, HTML, and markdown preview that updates as Claude responds
- **Code Workspace** — VS Code-inspired editor with file explorer, integrated terminal, debug panel, and AI assistant dock
- **Connection Management** — Configure multiple API keys, models, and parameters
- **Benchmarking** — Measure TTFT, throughput, render time, and Web Vitals

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 and add your Anthropic API key in Settings.
